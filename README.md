# Licenta-Capilnas.Matei
Descrierea aplicatiei desktop
"Aplicatei de gestionare a cheltuirlilor personale cu analiza grafica"
Aplicatia permite utilizatorul sa-si gestioneze veniturile si cheluielile, sa vizualizeze analize grafice si sa stabileasca bugete personale.
Functionalitati principale pentru utilizator:
1.	Autentificare si profil
Creare cont/autentificare (email+parola) phpmailer
Vizualizare si ediatre profil 
2.	Dashboard 
Vizualizarea balanta totala (diferenta intre venituri si cheltuieli)
Venituri si cheltuieli din luna curenta
Top categorii de cheltuieli
Grafic mic cu evolutia cheltuielilor
3.	Tranzactii 
Vizualizare lista cu toate tranzactiile
Adaugare tranzactie noua (tip,suma, categorie, data, descriere)
Editare/stergere tranzactie
Filtrare dupa perioada, categorie sau tip(venit/cheltuiala)
4.	Analiza grafica
Vizualizare pe grafice (PIE – distributia cheltuielilor pe categorii, BAR- cheltuieli lunare, LINE- evolutia balantei in timp)
Comparare perioade (saptamana/luna/an)
5.	Bugete
Setare buget pentru fiecare categorie
Afisare progres
Avertizare vizuala daca se depaseste limita
6.	Cardurile mele
Buton „+Adauga card”
Lista de carduri conectate
Status sincronizare
Buton „ Actualizeaza tranzactiile”
7.	Setari
Alegere moneda
Resetare de date si logout
Resetare date sau logout
