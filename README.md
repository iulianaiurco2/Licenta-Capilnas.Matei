# Licenta-Capilnas.Matei
Aplicația permite gestionarea veniturilor și cheltuielilor, vizualizarea balanței și a analizelor grafice, precum și stabilirea de bugete personale. Utilizatorul poate adăuga tranzacții, seta limite, conecta carduri bancare și personaliza setările, având mereu o imagine clară asupra situației sale financiare
